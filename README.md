## Getting Started



### Prerequisites

* Python 3.8 or greater
* Git

### Installation

1. Clone the repo
```sh
git clone https://ThorlabsSpectralWorks@dev.azure.com/ThorlabsSpectralWorks/tar-qc-recognition/_git/tar-qc-recognition
```

2. Navigate into the repo folder and pull the latest code from the development branch
```sh
$ git pull origin dev
```

3. Create a new virtual Python environment called `env`
```sh
$ python -m venv env
```

4. Activate the virtual environment.
    * Windows:
      ```sh
      $ ./env/Scripts/activate
      ```
    * Mac/Linux:
      ```sh
      $ source ./env/bin/activate
      ```

5. Install the required dependencies.
  ```sh
  $ python -m pip install -r requirements.txt
  ```

6. Deactivate the virtual Python environment when finished. (Windows and Mac/Linux)
```sh
$ deactivate
```
